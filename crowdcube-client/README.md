# CrowdCube - Crowdfunding Platform

## Key Features

- **User Authentication**: Secure login and registration with Firebase authentication including Google sign-in
- **Campaign Management**: Users can create, update, and delete their own crowdfunding campaigns
- **Smart Filtering**: View running campaigns with automatic deadline validation and sorting by donation amount
- **Donation System**: Support campaigns with integrated donation tracking and deadline enforcement
- **Responsive Design**: Fully responsive interface built with Tailwind CSS and DaisyUI for all devices
- **Private Routes**: Protected pages ensuring secure access to user-specific features and campaign management

## Technologies Used

- React.js
- React Router
- Firebase Authentication
- Tailwind CSS + DaisyUI
- Axios
- React Hot Toast
- Node.js + Express
- MongoDB
